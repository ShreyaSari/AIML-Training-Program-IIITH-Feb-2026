## Basic Data Visualization using Python

## Overview  
This notebook focuses on understanding and implementing basic data visualization techniques for exploratory data analysis (EDA). The goal of this lab is to analyze relationships between different features in an automobile dataset and extract meaningful insights using various plots.

The notebook demonstrates how visualizations help in identifying patterns, correlations, and trends that are essential for data-driven decision-making.

---

## Objectives  

- Understand the importance of data visualization in machine learning  
- Learn how to use different plotting techniques using Seaborn and Matplotlib  
- Analyze relationships between features such as price, horsepower, and fuel efficiency  
- Identify correlations and patterns in the dataset  
- Interpret visual insights effectively  

---

## Dataset  

The dataset used in this notebook is an automobile dataset containing various features related to cars.

**Key Features:**

- **Price** → Target variable representing the cost of the car  
- **Horsepower** → Engine power of the vehicle  
- **Engine Size** → Size of the engine  
- **Highway MPG** → Fuel efficiency on highways  
- **Body Style** → Type/design of the car  

---

## Key Concepts Covered  

### 1. Data Visualization Techniques  

- **Scatter Plots** → Analyze relationships between numerical variables  
- **Box Plots** → Understand distribution and detect outliers  
- **Violin & Swarm Plots** → Visualize distribution in detail  
- **Joint Plots** → Study correlation between variables  
- **Subplots** → Compare multiple visualizations together  
- **Heatmap** → Understand feature correlations  

---

### 2. Correlation Analysis  

- **Positive Correlation** → Both variables increase together  
- **Negative Correlation** → One increases while the other decreases  

Helps identify key factors affecting price.

---

### 3. Feature Relationships  

- Price vs Horsepower  
- Price vs Engine Size  
- Price vs Highway MPG  
- Price vs Body Style  

These relationships help understand pricing trends.

---

### 4. Data Distribution  

- **Histograms** → Show overall data spread  
- **Boxplots** → Highlight variation and outliers  
- **Violin plots** → Show density and distribution shape  

---

### 5. Comparative Visualization  

Subplots are used to visualize multiple graphs together for better comparison and understanding.

---

## Results and Insights  

- Car price is strongly influenced by horsepower and engine size  
- Higher fuel efficiency (highway-mpg) generally means lower price  
- Convertibles and hardtops are usually more expensive  
- Most cars fall in the low to mid-price range  
- Feature correlations highlight key pricing factors

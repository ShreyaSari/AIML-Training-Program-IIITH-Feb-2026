# AIML Training Program – Feb 2026

## IIIT Hyderabad

This repository contains all lab work, assignments, assessments, and projects completed as part of the **Artificial Intelligence and Machine Learning (AIML) Training Program** conducted by IIIT Hyderabad.

The repository covers practical implementations, experiments, and analytical studies across various machine learning and artificial intelligence domains including feature engineering, dimensionality reduction, manifold learning, computer vision, and customer analytics.

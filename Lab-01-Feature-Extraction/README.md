# Lab 01 – Feature Extraction

## 1. Overview
This lab focuses on understanding and implementing fundamental feature extraction techniques used in Artificial Intelligence and Machine Learning workflows. The primary objective is to transform raw data into structured numerical representations suitable for model development.

---

## 2. Objectives
- To understand the role of feature extraction in machine learning pipelines.
- To implement standard feature engineering techniques.
- To analyze how different representations impact downstream tasks.

---

## 3. Methodology
The lab involves preprocessing raw data followed by applying appropriate feature extraction techniques. Depending on the dataset, this includes:

- Text representation using techniques such as:
  - Bag of Words (BoW)
  - TF-IDF
- Basic data preprocessing and transformation
- Visualization and analysis (where applicable)

---

## 4. Tools and Technologies
- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Matplotlib (if applicable)

---

## 5. Repository Contents
- `Lab1_Feature_Extraction.ipynb` – Jupyter/Colab notebook containing implementation, code execution, and observations.

---

## 6. Learning Outcomes
Upon completion of this lab, the following competencies were developed:

- Understanding of feature engineering fundamentals.
- Practical implementation of structured feature representations.
- Insight into preparing datasets for machine learning models.
- Improved ability to analyze and interpret feature matrices.
